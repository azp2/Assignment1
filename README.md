# Student Management System

## Overview
A simple Java console application for managing students and grades.

## How to Run
1. Open the project in Eclipse or IntelliJ.
2. Ensure JDK is installed.
3. Run the `StudentManagemmentApp` class.
4. Use the menu options displayed in the console.

## Main Classes

### Student
- Stores student information.
- Manages marks.
- Calculates average and grade.

### StudentManagemmentApp
- Main application class.
- Handles user interaction and menu operations.

## Features
- Add a new student.
- Prevent duplicate student IDs.
- Add marks (0–100 validation).
- Display all students.
- Search student by ID.
- Calculate average and grade.
- Show student with the highest average.
